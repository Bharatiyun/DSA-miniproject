DESCRIPTION

This program manages student data in a mess. It add records, edit all details of the student, calculate the no. of absent and present days,calculate monthly bill and can be used by both an administrator and a student.

The code is divided into 3 files
- minipro.c which contains all the function definitions.
- minipro.h which contains all function prototype and structure declaration.
- mess.c which contains the main() function.
